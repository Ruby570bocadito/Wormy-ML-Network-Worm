# Kernel Modules (Skeletons)

These are C source skeletons provided as reference for future development.
They are NOT production-ready and have no build chain.

- `linux/rootkit.c` — LKM rootkit skeleton
- `linux/ebpf_hook.c` — eBPF hook skeleton
- `windows/driver.c` — Windows driver skeleton

Requires kernel headers, build toolchain, and signing infrastructure to compile.
